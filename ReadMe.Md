![](mse6.png)

l0la is a sidecar cli utility to monitor vital stats of other processes on OSX at development
time. It combines the output of `lsof`, `ps` and `netstat`, then monitors these values
while active.

## Up and running

### Homebrew
```
brew tap simonmittag/cli &&
  brew install l0la &&
  l0la -v
```
